# Number Guessing Game 

## Project Overview

The Number Guessing Game is a basic game that you play on your computer. You have to guess a number and you only get to try a few times. After each guess the game tells you if your guess is too high or too low. The Number Guessing Game is really easy to use. It is a great way for people who are new to programming to learn about things like loops and conditionals and how to get input from the user. The Number Guessing Game is a game for beginners because it is interactive and it helps people understand basic programming concepts like loops, conditionals and user input, in the Number Guessing Game.

The main purpose of this project is to:

## How to Play

1. Run the program using a C compiler.

2. When you start the game it will show you a message and tell you how many attempts the game will give you. The game will also show you the number of attempts you have to play the game.

3. Enter your guesses one by one.

4. The game is going to help you out by letting you know if the number you are thinking of is bigger or smaller, than the one you guessed. The game will tell you to guess a number or a smaller number. This way the game will guide you to find the number.

5. If you guess the number correctly within the allowed attempts you will win the game.. If you do not guess the secret number correctly then the secret number will be revealed to you. This is what happens when you do not guess the number within the allowed attempts, for the secret number.
